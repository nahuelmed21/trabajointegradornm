# Mi proyecto Web (cuatrimestral)

## Descripción
Este proyecto consiste en una página web dedicada a la banda de rock argentina Cielo Razzo. La web brinda información sobre su historia, discografía, imágenes de sus álbumes, detalles del último disco y novedades sobre eventos.

## Contenido
- Favicon realizado de la primer imagen representativa que recuerdo.
- Menú principal - biografía, discografía(considerados discos), galería(imágenes .png) y novedades.
- Biografía – Historia de la banda desde su formación hasta la actualidad.
- Discografía – Tabla general de todos los álbumes con enlaces para escuchar cada uno.
- Galería – Imágenes de las tapas de los discos.
- Detalle del último disco – Tabla con nombre de canciones, duración y número de canciones.
- Video destacado – Reproducción de un tema de la banda(mp4) usando controls.
- Novedades y eventos – Información sobre próximo show en Reconquista, junto con iframe(mapa del kiosko que vende entradas para el mismo), enlaces útiles y formulario para recibir noticias de la banda.
- Dato curioso – Información adicional sobre presentaciones de la banda en Reconquista.
- Footer – Enlaces a redes sociales y navegación rápida.

## Tecnologías utilizadas
- HTML5 para la estructura y el contenido de la página.
- CSS3 para el diseño visual y la adaptación responsive.
- Google Fonts (Montserrat) para la tipografía principal.
- Google Maps Embed para la integración del mapa interactivo.
- Etiquetas multimedia de HTML5 (video) para la reproducción de contenido audiovisual.
- Media Queries para mejorar la visualización en dispositivos móviles y tablets.

## Mejoras visuales incorporadas
- Implementación de una paleta de colores oscura inspirada en la estética del rock.
- Uso de variables CSS (:root) para facilitar la administración de colores y estilos.
- Diseño responsive mediante Flexbox, Grid y Media Queries.
- Animación de aparición en el título principal mediante @keyframes.
- Efectos hover en enlaces, botones y tarjetas de álbumes para mejorar la interacción del usuario.
- Sombras y bordes redondeados en secciones, imágenes, tablas y elementos multimedia.
- Galería de álbumes organizada con CSS Grid para una distribución adaptable.
- Centrado y adaptación responsiva del video y del mapa integrado.
- Formularios estilizados para mantener coherencia visual con el resto del sitio.
