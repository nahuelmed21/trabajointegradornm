# Mi proyecto Web (cuatrimestral)

## Descripción
Este proyecto se va a tratar de una página web que brinda información sobre la banda de rock argentina Cielo Razzo.

## Contenido
- Menú principal
- Información general y/o datos de la banda. 
- Imágenes sobre tapas de disco.
- Videos sobre álbumes completos.
